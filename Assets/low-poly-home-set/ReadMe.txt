[[ ENG ]]

--- Description ---

This time I tried to model a whole low poly simplistic home set, featuring things you'll find on any house!
Yes, I know there are many things missing here, I'll try to release more smaller home sets featuring more furniture (to make more unique combos and to populate things more, such as an actual garage, garden, pool, or just more bigger rooms).

You will find "incomplete" models, or missing parts. These are not missing, in order to optimize it, I deleted clones of the same parts (such as drawers), you will have to manually clone these and adjust them yourself in order to setup them correctly, you can mix and match different parts in order to create distinct variations of the same too, the set is modular after all!

There are some textures you can use and modify the way you want (kind of like placeholders), along with 6 palettes (12x12) you can edit too. Some models do feature a second material, such as TVs or phones (palette and then screen texture).

As always, you may find some important things missing, in which case I encourage you to tell me on my itch.io page, so I can update the set with more models! (you can also send me suggestions to vnb.prods@gmail.com)

You're free to use every model the way you want for any commercial / non-commercial project. (comments and mentions are always welcome and certainly appreciated)

I shouldn't really need to clarify this, but just in case, remember you're not able to re-sell these models individually or as part of a model pack.

Please make sure to review the asset on itch.io if you found it useful, leave a comment there, along with any kind of suggestion (or problem you faced while using it) in order to improve it!

[[ ESP ]]

--- Descripción ---

En esta ocasión decidí modelar un set bajo la temática 'Home' (hogar)! Tengo planeado crear más sets complementando este, a modo de poder expandir la casa en algo más amplio y detallado (un garage apropiado, jardines, etc..).

Encontrarás modelos "incompletos", como escritorios sin cajones, que realmente no estan incompletos, simplemente decidí eliminar clones del mismo objeto, deberás clonarlos y ajustarlos por tu cuenta (no es tan difícil, y puedes combinarlos para crear variantes distintas, dado que este es un set modular!).

El set utiliza una paleta de 12x12 pixeles (junto a otras 5 variantes), que puedes modificar para crear nuevas paletas. Algunos modelos fueron diseñados de forma que puedan manipularse de manera sencilla y animarse por separado, no deberías de encontrar ningún problema en termino de ajuste de pivot ni nada por el estilo, si quieres una puerta que gire apropiadamente, solo colocala donde quieras y listo (es recomendable utilizar snapping para mayor precisión).

Como siempre, es posible que encuentres que faltan otros objetos, por lo que te recomiendo realizar cualquier sugerencia en mi página de itch.io (tambien puedes enviar sugerencias a vnb.prods@gmail.com)

----------
Creditos a VNB (Leo) (https://vnbp.itch.io/)
